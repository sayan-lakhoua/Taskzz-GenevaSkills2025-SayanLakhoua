# Taskzz - Sayan Lakhoua

![Taskzz interface](src/logo-img/taskzz-logo.svg)

## Présentation
Taskzz est une application React qui centralise la création, l'organisation et le suivi de rappels personnels. Chaque action est synchronisée avec l'API Taskzz afin de retrouver ses tâches sur n'importe quel appareil.

## Où trouver l'application ?
- Code source complet : https://github.com/sayan-lakhoua/Taskzz-GenevaSkills2025-SayanLakhoua
- Déployé sur : https://2502.ict-expert.ch/

## Fonctionnalités principales
- Création, modification, archivage et suppression de tâches détaillées.
- Listes personnalisées avec codes couleur pour regrouper ses projets.
- Filtres par priorité, statut et échéance pour retrouver rapidement une information.
- Interface bilingue (FR/EN) avec mémorisation de la langue choisie.
- Authentification sécurisée et synchronisation en temps réel via l'API distante.
